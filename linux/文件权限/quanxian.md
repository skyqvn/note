# 文件权限

### 文件的权限针对三类对象进行定义
owner 属主，缩写u  
group 属组，缩写g  
other 其他，缩写o  

### 每个文件针对每类访问者定义了三种主要权限
r：Read 读  
w：Write 写  
x：eXecute 执行  

#### 文件与目录不同

|      | r                    | w                  | x                                              |
| ---- | -------------------- | ------------------ | ---------------------------------------------- |
| 文件 | 可以读取文件内容     | 可以修改文件内容   | 可以执行                                       |
| 目录 | 可以查看目录下的文件 | 可以删除或创建文件 | 可以cd进入目录，查看目录中文件属性（基础属性） |

### 修改权限方法

#### mode法

chmod who opt per file

> who: u,g,o,a(all)  
> opt: +,-,=  
> per: r,w,x  

#### 数字法

chmod xxx file

> r:4  
> w:2  
> x:1
